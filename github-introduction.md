- git
- GitHub
- remote
- local
- branch
- clone
- repo (repository)

## GitHub Commands
- git status
- git add <file-name>
- git commit -m "meaningful message"
- git push origin main

## Keyboard Shortcuts
- command + s = save
- commmand + r = refresh
- command + x = cut
- command + c = copy
- command + v = paste
- command + z = undo

## Additional Git Commands
- git log = show past commits
- git log --graph = Show a graphical representation of past commits
